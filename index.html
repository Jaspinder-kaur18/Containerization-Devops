🎯 Objective

To design and deploy a production-ready containerized web application using Docker, PostgreSQL, and advanced networking with Macvlan/Ipvlan. This project demonstrates real-world DevOps practices including orchestration, networking, and optimization.

🧰 Technology Stack

  • Database: PostgreSQL
  • Backend: Node.js + Express
  • Containerization: Docker (Multi-stage builds)
  • Orchestration: Docker Compose
  • Networking: Macvlan / Ipvlan

🏗️ Architecture

Client (Browser/Postman)
⬇
Backend Container (Static IP)
⬇
PostgreSQL Container (Volume + Static IP)
  • Containers have LAN IPs
  • Backend communicates with DB internally
  • Persistent storage via volumes

🔗 API Endpoints

  • POST /data → Insert record
  • GET /data → Fetch records
  • GET /health → Healthcheck
app.post('/data', ...); app.get('/data', ...); app.get('/health', ...);

🐳 Docker Implementation

  • Multi-stage builds used
  • Alpine base image
  • Non-root user security
  • .dockerignore implemented
FROM node:18-alpine AS builder WORKDIR /app COPY package*.json ./ RUN npm install COPY . . FROM node:18-alpine WORKDIR /app COPY --from=builder /app . CMD ["node","server.js"]
